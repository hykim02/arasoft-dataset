업무개요

□ 구정 사업 내용 홍보를 위한 홈페이지용 배너이미지 제작 및 게시

업무 프로세스

 홍보 신청 공문 접수

- 웹사이트 배너 신청서.hwp

 배너 이미지 제작

- 홈페이지 용역업체에 업무 게시판을 통하여 이미지 제작 요청

 <게시판 작성 내용>

- 배너 제목 및 본문 내용 작성

- 참고 이미지 전달(사업 포스터 등)

- 크기 지정(대표홈페이지/보건소 , 메인/서브 배너 등 각 배너별로 이미지 상이함)

 홈페이지 관리시스템 로그인

 콘텐츠 ⇒ 대표이미지 관리⇒ 등록

  - 배너 제목 입력

- 사이트 선택 : 구청 홈페이지/태안서초휴양소 홈페이지/보건소 홈페이지 등 

                         게시할 홈페이지 선택

- 적용기간 선택 : 게시 시작 날짜 및 시간부터 게시 종료 날짜 및 시간까지 

                             선택 가능

- ALT 작성 : 시각장애인을 위한 음성지원 기능을 위해 배너이미지의 내용과 

                     동일한 내용을 텍스트로 입력

- 타겟 : 현재창/새창 (새창선택시 배너를 클릭하면 새창에서 링크로 이동)

- 링크 : 배너를 클릭하면 이동할 홈페이지 주소 입력

- 배너 이미지 파일 선택

- 진행형태 : 상시게시/사용중/종료 중 선택가능
