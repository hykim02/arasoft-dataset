■ 이호조 등록 방법

1. 이호조 → 지출관리 → 지출품의 → 지출품의결재요청 제목란에 제목 입력 → 예산과목 추가 클릭 → 지출사업목록생성 → 세부사업명(쾌적한민원실운영 선택) → 통계목(공공운영비 선택) → 품의요청액 입력 → 결재요청(하단)


2. 기안문 작성 팝업창이 뜸 제목/본문에 기안문 작성 / 첨부파일 첨부하기 → 하단 기안전송


3. 전자결재 행정정보연계 → 기안 (내부결재 협조 없음) → 결재완료


4-1 결재완료 되면 이호조 지출품의승인내역 해당건 클릭 지출품의승인내역 등록 → 의무/재량구분 : 법정 → 품의유형(일반) → 공통정보 집행내용란에 문서대호 입력 → 지출금액입력


4-2 → 거래처 추가 → 거래처계좌번호 거래처명 : 양재동우체국 입력 선택 ->추가 거래처명 : 서초우체국 입력-> 거래처 란에 양재/서초 2건이 보이면 성공// 거래처구분 법인-> 입금유형: 고지서 -> 공급가액 : 지출금액 각각 입력하기 -> 결재 공문 첨부하기(중요 : 결재문서는 공개로 저장) - > 붙임문서 제대로 붙어 있는지 확인 -> 원인행위등록요청


4-3 지출결의 하기 -> 원인행위완료(재무과)되면 지출결의요청 팝업이 뜸 → 지출결의 → 지출결의요청등록 → 해당건 선택 → 지급유형 (일반지출) → 증빙구분 기타 선택 → 분개정보 우측 다중분개 클릭 → 업무코드분류 (일반지출 선택 → 업무코드명(우편요금검수 선택) → 승인요청

업무개요

□ 법정처리기간 7일 이상인 유기한 민원 대상으로 민원후견인을 지정하여 신속·공정한 민원 업무처리 지원

업무 프로세스

 후견인 지정 요청

- 민원인이 후견인 지정 요청

▶ 민원후견인 명단 : 총 6명 (오케이민원센터장, 민원행정팀장, 

    가족관계팀장, 제증명팀장, 인허가팀장, 여권팀장)

▶ 민원대행업체(건축사, 법무사 등)가 수행하는 민원 제외

 후견인 지정 및 활동수행

- 해당 유기한민원 접수시 후견인 지정 및 통보

- 후견인 등록방법 : 새올 ‘민원’ → ‘민원처리’ → ‘민원신청등록’ → ‘민원명 입력’ 

   → ‘후견인(검색, 선택)’ → 민원접수

    ▶ 민원인에게 후견인 지정 안내서 교부

 - 민원처리과정 및 결과안내(후견인→민원인)

 후견활동 결과보고

- 후견인활동일지 새올입력

   ▶ 민원종료 3일 이내

- 새올 ‘민원’ → ‘민원처리’ → ‘후견활동대상민원조회’ → 접수기간 선택, 

   조회 → ‘후견인활동상황처리’ → ‘추가’ 버튼을 누르고 활동사항 입력 

   → 저장 후 닫기
