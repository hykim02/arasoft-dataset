 응답소 현장민원 접수 확인

- 응답소(120)에서 서초구 업무담당자에게 배분한 현장민원 확인(새올을 통해 

  응답소 프로그램 접속, 수시 확인) 

※ 응답소 기관관리자 권한 필요(권한 요청은서울시 민원담당관으로 공문 제출)

 응답소 현장민원 처리 확인

- 처리 지연 시 : 담당자 처리 독려

- 담당자 재지정 요청 시

    ∘ 담당자 재지정 : 민원업무 → 민원회수/처리자 재분배 → 민원 접수번호 

       입력 후 저장(기관 재지정은 서초구 외 다른 기관 이송)

    ∘ 협조부서 요청 : 응답소 협조요청 관리 → 민원 접수번호 입력, 조회 → 

       민원 협조부서 선택 후 저장


※ 현장민원의 경우 담당자 전결로 처리 (전자결재 생략)
