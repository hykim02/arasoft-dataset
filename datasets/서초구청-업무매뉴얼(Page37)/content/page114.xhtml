■ 이호조 등록 방법

1. 이호조 → 지출관리 → 지출품의 → 지출품의결재요청 제목란에 제목 입력 → 예산과목 추가 클릭 → 지출사업목록생성 → 세부사업명(쾌적한민원실운영 선택) → 통계목(공공운영비 선택) → 품의요청액 입력 → 결재요청(하단)


2. 기안문 작성 팝업창이 뜸 제목/본문에 기안문 작성 / 첨부파일 첨부하기 → 하단 기안전송


3. 전자결재 행정정보연계 → 기안 (내부결재 협조 없음) → 결재완료


4-1 결재완료 되면 이호조 지출품의승인내역 해당건 클릭 지출품의승인내역 등록 → 의무/재량구분 : 법정 → 품의유형(일반) → 공통정보 집행내용란에 문서대호 입력 → 지출금액입력


4-2 → 거래처 추가 → 거래처계좌번호 거래처명 : 양재동우체국 입력 선택 ->추가 거래처명 : 서초우체국 입력-> 거래처 란에 양재/서초 2건이 보이면 성공// 거래처구분 법인-> 입금유형: 고지서 -> 공급가액 : 지출금액 각각 입력하기 -> 결재 공문 첨부하기(중요 : 결재문서는 공개로 저장) - > 붙임문서 제대로 붙어 있는지 확인 -> 원인행위등록요청


4-3 지출결의 하기 -> 원인행위완료(재무과)되면 지출결의요청 팝업이 뜸 → 지출결의 → 지출결의요청등록 → 해당건 선택 → 지급유형 (일반지출) → 증빙구분 기타 선택 → 분개정보 우측 다중분개 클릭 → 업무코드분류 (일반지출 선택 → 업무코드명(우편요금검수 선택) → 승인요청

업무개요

□ 응답소 일반민원 처리부서 지정 및 120 다산콜센터 민원 상담을 위한 표준상담

   데이터베이스(DB) 업데이트 실시

업무 프로세스

  <<응답소 일반민원>>

     응답소 일반민원 접수 확인

- 응답소에서 서초구로 배분한 ‘기관분배’ 민원 확인(새올을 통해 응답소 

   프로그램 접속, 수시 확인)

▶ 응답소 기관관리자 권한 필요(권한 요청은 서울시 민원담당관으로 공문 제출)

 응답소 일반민원 부서 분배

- 서초구로 기관분배된 해당 민원 클릭

- 처리부서 선택 후 저장 

▶ 부서 재지정 또는 협조부서 요청시 처리방법

 ∘ 부서 재지정 : 재지정 요청된 해당 민원 클릭 후 처리부서 재지정 선택하여 저장

    (기관 재지정은 서초구 외 다른 기관 이송)

 ∘ 협조부서 요청 : 응답소 ‘협조요청 관리 → 민원 접수번호 입력, 조회 → 

    민원 협조부서 선택 후 저장

<<120 표준상담 DB 관리>>

 표준상담 DB 접속

   - 담당자가 표준상담 DB 수정한 경우 승인필요

   ▶ 새올 응답소 접속(민원지식 → 120 표준상담 DB 관리 → 승인관리)

 표준상담 DB 수정 승인

  - 인사이동에 따른 담당자 변경 등 표준상담 DB 수정시 각 담당자가 변경사항을

     입력하고 저장하면 기관관리자가 최종적으로 승인하여 변경 확정됨


  <<응답소 사용자 권한 관리>>

    ▶ 응답소 사용자 권한 신청 확인

  - 응답소 ’민원운영‘ → ’응답소 사용자 권한 관리‘에서 권한 신청 내역 

     수시 확인 → 부서관리자(보통 서무주임 변경시) 권한 승인(해제) 처리
