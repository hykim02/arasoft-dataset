■ 이호조 등록 방법

1. 이호조 → 지출관리 → 지출품의 → 지출품의결재요청 제목란에 제목 입력 → 예산과목 추가 클릭 → 지출사업목록생성 → 세부사업명(쾌적한민원실운영 선택) → 통계목(공공운영비 선택) → 품의요청액 입력 → 결재요청(하단)


2. 기안문 작성 팝업창이 뜸 제목/본문에 기안문 작성 / 첨부파일 첨부하기 → 하단 기안전송


3. 전자결재 행정정보연계 → 기안 (내부결재 협조 없음) → 결재완료


4-1 결재완료 되면 이호조 지출품의승인내역 해당건 클릭 지출품의승인내역 등록 → 의무/재량구분 : 법정 → 품의유형(일반) → 공통정보 집행내용란에 문서대호 입력 → 지출금액입력


4-2 → 거래처 추가 → 거래처계좌번호 거래처명 : 양재동우체국 입력 선택 ->추가 거래처명 : 서초우체국 입력-> 거래처 란에 양재/서초 2건이 보이면 성공// 거래처구분 법인-> 입금유형: 고지서 -> 공급가액 : 지출금액 각각 입력하기 -> 결재 공문 첨부하기(중요 : 결재문서는 공개로 저장) - > 붙임문서 제대로 붙어 있는지 확인 -> 원인행위등록요청


4-3 지출결의 하기 -> 원인행위완료(재무과)되면 지출결의요청 팝업이 뜸 → 지출결의 → 지출결의요청등록 → 해당건 선택 → 지급유형 (일반지출) → 증빙구분 기타 선택 → 분개정보 우측 다중분개 클릭 → 업무코드분류 (일반지출 선택 → 업무코드명(우편요금검수 선택) → 승인요청

업무개요

□ 구에서 접수한 민원사무 처리현황을 확인·점검하여 신속 공정한 민원처리 도모 

   (매월 1회)

업무 프로세스

 민원데이터 추출(새올, 응답소) 

 - 새올 유기한민원 데이터 추출
(새올 ‘민원’ → ‘민원처리’ → ‘민원접수 및 처리내역 관리’ → ‘민원처리부’ 출력)

 - 응답소 일반민원 데이터 추출방법 (새올 ‘응답소’ → ‘민원조회’ → ‘총괄민원 

    조회’ → 구분 ‘일반민원’ 선택 후 엑셀 저장)

 - 민원창구 발급데이터 추출 (제증명, 가족관계, 여권팀을 통해 발급통계 확인)

 - 무인민원발급기 데이터 추출 (새올 ‘민원’ → ‘무인민원발급창구’ → ‘발급통계

    관리’ → ‘발급기별 통계조회’ → 조회기간 선택 후 검색)

 보고서 작성

- 월 단위 민원 처리상황 및 운영실태 점검 보고서 작성 (결재자 : 구청장)

※ 전월 대비 비교 증감 데이터가 많이 들어가므로 전월 보고문서 불러오기 해서 기안하면 편리(Tip)

- 부서별 스피드지수 평가자료 작성 (자치구 평가자료 업로드 사이트 접속, 

   엑셀자료 저장 : 자치구 민원정보), 부서별로 재분류 후 순위 산출)

- 새올 및 응답소 점검결과 작성 (새올 ‘민원처리부’에서 지연처리 데이터 추출 /

   응답소 총괄민원 데이터에서 지연처리 데이터 추출)

▶ 민원 처리 지연자는 감사담당관으로 공문 제출
  (단, 응답소 1일 미만 지연건은 미포함)

- 만족도 평가자료 작성 (새올 ‘응답소’ → ‘민원조회’ → ‘국민신문고만족도

  모니터링’ → ‘처리일자’ 선택 후 엑셀저장)

- 보고서 결재 후 전부서 수신자로 지정해서 점검결과 알림 공문 시행 

  (민원 처리 지연부서는 민원처리 자체교육 실시 후 결과 제출 안내)
