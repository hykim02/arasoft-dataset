업무개요

□ 행정기관 소속 공무원이 행정정보공동이용, 정부24 등의 정보시스템에서 사용자 로그인을 통한 신원 확인, 송수신 데이터 암호화 등에 활용하기 위해, 부처별 개인단위로 인증서를 발급

업무 프로세스 (발급/재발급)

 행정전자서명 신청서 작성 및 공문 발송 (신청인)

1.1. 행정전자서명 신청서 양식 다운로드 (경로 : 새올행정포털 접속 > 업무공지 > ‘전자서명’ 검색)

1.2. 행정전자서명 신청서 양식 작성

1.3. 신청인 및 부서장 서명 또는 날인

1.4. 공문에 신청서를 첨부하여 공문 발송 (수신처 : 스마트도시과) 

※ 스마트폰 촬영본 접수 불가 (스캔본 첨부)


 공문 접수 (헹정전자서명 담당자)

- 서초구 등록기관 담당자 (행정전자서명) 공문 접수


 인증관리자시스템 접속 및 신청자 확인·등록

(행정전자서명 담당자)

3.1. 인증관리자시스템 접속

3.2. 공문으로 접수된 신청자의 신원확인 실시

3.3. 사용자 관리 > 사용자 조회 메뉴에서 조회 후 등록 실시

3.4. 사용자 등록 시 인증코드(인증번호)가 자동 생성되며, 신청자에게 메시지 발송


 인증서 발급 / 재발급 (신청인)

4.1. 사용자(신청인)는 인증서 발급처리 완료문자 수신 후 인증관리센터 홈페이지 (https://www. gpki.go.kr) 접속하여 ‘발급’ 메뉴에서 인증서 발급 선택 (경로 : 인증관리센터 홈페이지 접속 > 인증서 관리 > 발급)

4.2. 구분 및 기관명 선택

- 구 분 : 개인용, 행정기관

- 기관명 : 서울특별시

※ 기관명 검색은 최상위기관 선택

4.3. 본인확인 수행 (이름,주민등록번호)

4.4. 입력정보 확인 (인증번호 입력)

4.5. 인증서 저장매체 선택 (하드디스크, 이동식디스크 등)

4.6. 인증서 암호 입력

4.7. 인증서 발급 완료 확인

업무 프로세스 (갱신)

 인증서 갱신 (신청인)

1.1. 인증관리센터 홈페이지 (https://www. gpki.go.kr) 접속하여 ‘발급’ 메뉴에 인증서 갱신 선택 (경로 : 인증관리센터 홈페이지 접속 > 인증서 관리 > 갱신)

1.2. 갱신할 인증서 선택한 후 ‘비밀번호’ 입력

1.3. 인증서 저장할 매체 선택

1.4. 인증서 갱신 완료 확인
